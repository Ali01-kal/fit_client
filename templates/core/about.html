About Our Project

fitClient Fitness CRM System

Клиенттер, тренерлер, бағдарламалар, абонементтер, пікірлер, API және admin панелі бір Django жобасында біріктірілген.

Бұл нұсқа Colorlib Zacson шаблоны негізінде интеграцияланып, `fitClient` деректерімен толтырылды.

Команда проекта и роли

  • Али (Ali01-kal) — Fullstack Developer (Backend, Frontend, API, Admin)
  • Командный формат ролей соблюдён: разработка, UI, API, администрирование и эксплуатация разделены по ролям
  • Текущая реализация — индивидуальный учебный проект, где один автор выполняет все роли команды
  • Роль деплоя/домена/SSL и PostgreSQL на проде — выполняется автором при публикации на хостинге
